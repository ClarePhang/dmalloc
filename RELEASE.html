Debug Malloc (Dmalloc) Library Release Notes

Version: Release 5.0.1
Last-Version: 5.0.0
Date: May 20, 2003

Overview    New Functionality    Dmalloc Utility    settings.dist    Installation    Programming    Environmental Variables    Bugs Fixed

Overview

Bug fix release after the big 5.0.0 last week. Couple of bug fixes and improvements.

New Functionality

  • Added dmalloc_page_size() function which returns the internal library page/block size.
  • Compressed the per-pointer admin structure by 8 bytes.

Dmalloc Utility Changes

None.

Changes to the settings.dist File

None.

Installation Changes

None.

Programming Notes

  • Reduced the per-pointer administrative overhead by 8 bytes down to ~32 bytes on an i386.
  • Added a number of auto-testing sections to dmalloc_t. Still need to flesh this out to be more inclusive of library features.
  • Added testing of the free-list during heap-check.

Environmental Variable Changes

None.

Bugs Fixed

  • Found a [bad] problem where the library was not putting pointers back on the free list properly. Ugh.
