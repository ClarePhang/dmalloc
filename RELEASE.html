Debug Malloc (Dmalloc) Library Release Notes

Version: Release 5.2.3
Last-Version: 5.2.2
Date: September 29, 2003

Overview    New Functionality    Dmalloc Utility    settings.dist    Installation    Programming    Environmental Variables    Bugs Fixed

Overview

Added some better handling for Cygwin environments. Other fixes.

New Functionality

None.

Dmalloc Utility Changes

None.

Changes to the settings.dist File

None.

Installation Changes

  • Added a section to the configure script to test for the existance of the GetEnvironmentVariableA for Cygwin environments which did not like getenv().

Programming Notes

  • Added new constructor and GetEnvironmentVariableA tests to better initialize the library.
  • Added better generation of the dmalloc.h include file.

Environmental Variable Changes

None.

Bugs Fixed

  • Removed usage of the environ variable from dmalloc_argv.c. It is usually an exported variable.
  • Changed the configure script to define the HAVE_ variables as 1 or 0 to better satisfy compilers.
